Annotate a Phosphoproteomic Dataset

Upload a CSV or TSV file, identify the UniProt accession and peptide columns, choose annotation options, and download the result enriched with gene names, domains, GO terms, and site-level information from ProteomeScout.

1Upload Dataset

Maximum file size: 100 MB. The file must contain a header row with column names.

2Select Columns

Peptides must have modified residues written in lowercase (e.g. ANsFVGTAQYV for phospho-S241). All other residues should be uppercase.

3Annotation Options

4Annotate & Download

Annotation queries ProteomeScout for every row in your dataset. Processing time scales with dataset size — a few hundred rows typically completes in under a minute.
